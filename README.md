How to config environment for Ruby on Rails

1) Tools for RoR
- About database:
    + DB Browser for SQLite
    + Sequel Pro
- About Editor:
    + VS Code (Install debug)
    + NetBean
    + Atom
- Git hub account
    + Source tree
- FileZilla

2) Install package library for RoR
- Install mysql
    + brew install mysql
- Install SQLite version 3.x
    + gem install sqlite3
- Create database
    + rake db:create
    + rake db:migrate
    + rake db:seed
